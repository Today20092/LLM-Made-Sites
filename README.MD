# LLM Made Sites

Astro migration of the static tool collection.

## Routes

- `/` hub
- `/prompt-launcher`
- `/video-bitrate-calculator`
- `/md-to-pictures`

## Stack

- Astro
- Tailwind via `@astrojs/tailwind`
- `marked`
- `html2canvas`

## Local dev

```bash
npm install
npm run dev
```

## Build

```bash
npm run build
```
